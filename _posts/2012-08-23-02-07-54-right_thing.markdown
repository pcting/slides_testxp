<div class="right" markdown="1">
### What not to do
* Don't test impossible cases
* Don't test untestable code
* Do not test 3rd party services/libraries
<br />
### What to do
* Ensure correctness!
* Test core behavior
* Setup clean environment per test
* Mock 3rd party services/libraries
</div>
